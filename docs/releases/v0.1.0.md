# Gpt Orbit v0.1.0

Gpt Orbit Weekly 的首个公开版本。这是一个面向 Windows 11 的轻量桌面悬浮球，用来快速查看 Codex 周额度。

## 主要功能

- 一眼查看本周剩余额度百分比。
- 显示距离周额度重置的倒计时。
- 用独立徽章显示剩余重置次数。
- 支持悬停展开、拖动定位、窗口置顶、托盘运行与位置记忆。
- 数据源断开时保留最近一次已知结果，并在界面中标明过期状态。

## 安装

1. 在本 Release 的 **Assets** 中下载 `Gpt.Orbit.Weekly_0.1.0_x64-setup.exe`。
2. 核对下方文件大小与 SHA-256。
3. 运行安装程序，然后保持官方 Codex 应用处于已登录状态。

系统要求：Windows 11 x64。

## 隐私说明

Gpt Orbit 从本机 Codex `app-server` 读取额度状态；周额度版还会使用本机 Codex 登录材料，通过对应的 OpenAI HTTPS 端点读取重置次数。项目不设置中转服务器，也不把 token、邮箱或完整响应写入日志或缓存。
