## 小红书

### 标题

我做了个 Codex 额度悬浮球

### 正文

每次想看 Codex 周额度，都要重新找入口，我干脆给自己做了一个 Windows 桌面悬浮球：Gpt Orbit Weekly。

现在不用打断手上的工作，一眼就能看到：

- 本周还剩多少额度
- 距离周额度重置还有多久
- 还剩几次额度重置
- 悬浮置顶、自由拖动、托盘运行和位置记忆

目前首发版支持 Windows 11 x64，安装包可以直接使用，源码也已经全部公开，免费开源。

GitHub：https://github.com/316380027-spec/gpt-orbit

安装包：进入仓库 Releases，选择 `v0.1.0`。

这是我个人制作的第三方开源小工具，与 OpenAI 没有官方关联。第一次公开发布，欢迎告诉我你最想补充什么功能，也欢迎反馈实际使用中遇到的问题。
