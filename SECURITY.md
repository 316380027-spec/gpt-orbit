# 安全政策

## 私下报告漏洞

请通过本仓库的 [GitHub Security Advisories 私密报告入口](https://github.com/316380027-spec/gpt-orbit/security/advisories/new) 提交安全漏洞：进入仓库的 **Security → Advisories**，选择 **Report a vulnerability**。报告及后续讨论只对报告者和仓库安全维护者可见。

请提供受影响版本、复现步骤、影响和建议修复，但只附最小化且已脱敏的证据。

## 不应公开或提交的内容

请勿在公开 Issue、Discussion、截图、日志或安全报告附件中提交：

- 登录 URL 或其中的参数；
- token、Cookie、授权请求头或其他凭据；
- 邮箱、账号标识、credit ID 等账户信息；
- 完整的 Codex `app-server` 请求或响应载荷。

如果公开问题可能涉及安全风险，请停止公开讨论，改用上述 Security Advisories 私密入口。
